CheatSheet for JS

Variables

JavaScript variables are containers for storing data values.
ex: var x = 5;
var y = 6;
var z = x + y;

Debugging (alerts, console, writing to HTML element)

Searching for errors in programming code is called code debugging.

-The console.log() Method to display JavaScript values in the debugger window
ex: a = 5;
b = 6; c = a + b;
console.log(c);

-Writing to HTML element To change the content of an HTML element
document.getElementById("p1").innerHTML = "New text!";

-Alert
An alert box is often used if you want to make sure information comes through to the user.
alert("Hello World!!");

Different data types

JavaScript variables can hold many data types: numbers, strings, arrays, objects.
ex: var length = 16 // Number
var lastName = "Johnson" // String
var cars = ["Saab", "Volvo", "BMW"]; // Array
var x = {firstName:"John", lastName:"Doe"}; // object

Arrays

JavaScript arrays are used to store multiple values in a single variable.
ex: var cars = ["Saab", "Volvo", "BMW"];
document.getElementById("demo").innerHTML = cars;

Objects

JavaScript objects are containers for named values (called properties) and methods.
ex: var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};

Testing

any test returns a boolean true or false
To test if two strings are equal:
ex: "stringone" === "string two"; >false
(10-5) == 5; (10-5) == "5";
"stringone" !== "string two"; >true
5 > 10; > false

Logic

-The if statement allows us to run code only if a certain test evaluates to true.
ex: if(5>10){ console.log("You'll never see this in the console because 5 is not greater than 10"); }
if(5>10){ console.log("You'll never see this because 5 is not greater than 10"); } else{ console.log("You will see this though"); }
